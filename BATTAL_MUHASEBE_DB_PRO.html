🏛️ Sabit Kıymet / Demirbaş Yönetimi

Kural: KDV hariç >12.000 TL fiziki/gözle görülür eşya → Demirbaş (amortismana tabi). <12.000 veya sarf/hizmet → normal gider.

Hizmet işletmesi (avukat/hekim/fotoğrafçı/müşavir): "Mal alışı" yoktur — alımlar sarf malzeme veya demirbaş olur.

Gider sekmesinde sabit kıymet otomatik yakalanır ve sınıflanır. Aktifleştirme amortisman akışı için Defter Beyan'da elle "Sabit Kıymet Yönetimi → Aktifleştirme" menüsünden devam edilir (faydalı ömür + sabit kıymet kodu seçimi).

Önce Gider sekmesinden CSV yükle — >12.000 TL demirbaşlar burada listelenir.

✅ Script Klibborda Kopyalandı

📋 Defter Beyan'a giden script hazır ve klibborda kopyalandı. Şimdi:
  1. Defter Beyan sekmesini aç (giriş yapmışsın) ve mükellefe geç:
  2. Klavyeden F12 → üstten "Konsol" sekmesine tıkla
  3. Konsol kapalıysa "Yapıştırmaya izin ver" diye yaz Enter → açılır
  4. Konsola Ctrl+V ile yapıştır → Enter
  5. Hepsi otomatik gider, başarılı/hatalı sayısı konsola düşer ✅

Script tarayıcında o anda token'ı ile çalışır, Defter Beyan API'sine direkt POST atar.

❓ Nasıl Kullanılır

  1. Üst yeşil "🔑 Token" butonuna tıkla, Defter Beyan'a giriş yapıp token'ı al ve yapıştır (bir kez)
  2. Mükellef kartlarından seç
  3. "Gelir / e-SMM" veya "Gider / e-Fatura" sekmesinden Uyumsoft Excel/CSV bırak
  4. Dönem (ay/yıl) seç → otomatik mevcut ay seçili
  5. Tabloda kontrol et: ✅ temiz / ⚠️ uyarı / ❌ iptal-mükerrer
  6. "🚀 Defter Beyan Scripti Üret + Kopyala" butonuna tıkla
  7. Açılan pencerede yönlendirmelere uy: Defter Beyan'a geç, F12 → Konsol → Ctrl+V → Enter
  8. Script API ile gönderir, hepsi otomatik kaydedilir

Mükerrer koruması: Defter Beyan aynı VKN + tarih + fatura no'yu ikinci kez kabul etmez. Bu yüzden script güvenli — yanlışlıkla 2 kez çalıştırsan da çift kayıt olmaz.
